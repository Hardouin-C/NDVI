
The attached file contains the code to create a real-world environmental dataset.

The study area is the central Pyrenees region, defined by the following coordinates:

North-West Corner: 42.90°N, 0°E

South-East Corner: 42.60°N, 0.4°E

The program allows for the download and visualization of the following variables:

NDVI (Normalized Difference Vegetation Index), Elevation, Slope, Southness. 

A data frame is generated based on these variables along with the latitude and longitude coordinates.

The results of a simple linear regression of NDVI upon the other variables are analyzed, highlighting the necessity of using a spatial regression model for this specific data.
